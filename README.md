# Vuer Skill

Claude Code skill for [Vuer](https://github.com/vuer-ai/vuer) - 3D visualization toolkit for robotics and GenAI.

## Installation

```
/plugin marketplace add vuer-ai/vuer-skill
/plugin install vuer@vuer-ai-vuer-skill
```

## What's Included

Skills for working with Vuer:
- **getting-started** - Quick start guide
- **components** - 3D components (Box, Sphere, TriMesh, PointCloud, URDF, etc.)
- **server** - Server configuration and session APIs
- **events** - Event handling and RPC methods
- **client** - Client-side usage
- **xr** - VR/AR/WebXR features
- **examples** - Common patterns and recipes
